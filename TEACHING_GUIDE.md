# คู่มือการสอน: การแทรกแผนที่ Google Maps (อย่างง่าย)

สำหรับให้นักศึกษาฝึกปฏิบัติ การนำแผนที่ Google Maps มาแสดงบนหน้าเว็บ (map.php) โดยไม่ต้องเขียนโค้ด JavaScript

## ขั้นตอนการทำ (Step-by-Step)

1. **เปิด Google Maps**
   - เข้าไปที่เว็บไซต์ [https://www.google.co.th/maps](https://www.google.co.th/maps)

2. **ค้นหาสถานที่**
   - พิมพ์ชื่อวัดที่ต้องการ เช่น "วัดแสนสุข ชลบุรี" ในช่องค้นหา

3. **กดปุ่ม "แชร์" (Share)**
   - เมื่อเจอสถานที่แล้ว ให้กดปุ่มไอคอน **"แชร์"** (หรือ Share) ที่เมนูของสถานที่นั้น

4. **เลือกแท็บ "ฝังแผนที่" (Embed a map)**
   - ในหน้าต่างที่เด้งขึ้นมา ให้เลือกแท็บที่เขียนว่า **"ฝังแผนที่"**
   - จะเห็นตัวอย่างแผนที่ และโค้ด `HTML` ที่ขึ้นต้นด้วย `<iframe ... >`

5. **คัดลอก HTML (Copy HTML)**
   - กดปุ่ม **"คัดลอก HTML"** เพื่อก๊อปปี้โค้ดทั้งหมด

6. **นำไปวางในไฟล์ map.php**
   - เปิดไฟล์ `map.php`
   - นำโค้ดที่คัดลอกมา ไปวางแทนที่ `<iframe>` เดิม หรือวางในตำแหน่งที่ต้องการแสดงแผนที่

## ตัวอย่างโค้ดที่ได้
```html
<iframe src="https://www.google.com/maps/embed?pb=!1m18!1m12!1m3!1d3875..." width="600" height="450" style="border:0;" allowfullscreen="" loading="lazy"></iframe>
```

---
*เคล็ดลับ: สามารถปรับค่า `width="100%"` เพื่อให้แผนที่ยืดเต็มความกว้างของหน้าจอได้*
